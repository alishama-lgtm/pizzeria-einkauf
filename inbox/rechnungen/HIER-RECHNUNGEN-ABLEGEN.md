# Rechnungen Inbox

Dateien hier ablegen → App übernimmt sie automatisch in den Verlauf-Tab.

## Unterstützte Formate
| Format | Erklärung |
|--------|-----------|
| `.pdf` | Lieferanten-Rechnung als PDF |
| `.jpg` `.jpeg` `.png` `.webp` | Foto der Rechnung (z.B. vom Handy) |
| `.xlsx` `.xls` | Excel-Rechnung |
| `.csv` | Tabellarische Rechnung |

## Dateiname-Format (empfohlen)
```
2026-04-14_Metro_245.50.pdf
2026-04-14_UM-Trade_312.00.jpg
14.04.2026_Etsan_89.90.xlsx
```
→ **Datum_Lieferant_Betrag**.Endung

Die App erkennt automatisch:
- **Datum** aus Dateiname (oder heute wenn nicht angegeben)
- **Lieferant** aus Dateiname
- **Betrag** aus Dateiname

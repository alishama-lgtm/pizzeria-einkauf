# Rechnungen Inbox

PDFs hier ablegen → App übernimmt sie automatisch in den Verlauf-Tab.

**Dateiname-Format (empfohlen):**
```
2026-04-14_Metro_245.50.pdf
2026-04-14_UM-Trade_312.00.pdf
2026-04-14_Etsan_89.90.pdf
```

→ Datum_Lieferant_Betrag.pdf

Die App erkennt automatisch:
- Datum aus Dateiname
- Lieferant aus Dateiname
- Betrag aus Dateiname

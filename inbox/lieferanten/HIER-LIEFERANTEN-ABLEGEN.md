# Lieferanten Inbox

CSV-Dateien hier ablegen → App aktualisiert Lieferanten-Tab automatisch.

**CSV-Format (Spalten):**
```
name,kontakt,telefon,email,kategorie,bewertung
Metro Wien,Hr. Müller,+43 1 234567,metro@example.at,Lebensmittel,5
UM Trade,Hr. Yilmaz,+43 699 123456,um@example.at,Fleisch,4
```

Erste Zeile = Kopfzeile (wird übersprungen).

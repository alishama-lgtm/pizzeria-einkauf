# Lieferanten Inbox

Dateien hier ablegen → App aktualisiert Lieferanten-Tab automatisch.

## Unterstützte Formate
| Format | Erklärung |
|--------|-----------|
| `.xlsx` `.xls` | Excel-Lieferantenliste (empfohlen) |
| `.csv` | CSV-Liste |
| `.json` | JSON-Export |
| `.pdf` `.jpg` `.png` `.webp` | Dokument (wird erkannt, kein Auto-Import) |

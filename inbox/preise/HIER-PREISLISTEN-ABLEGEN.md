# Preislisten Inbox

CSV-Dateien hier ablegen → App aktualisiert Produktpreise automatisch.

**CSV-Format (Spalten):**
```
produkt,preis,einheit,shop
Mozzarella 1kg,4.99,kg,Metro
Pizzamehl 25kg,16.90,Sack,Metro
Olivenöl 3L,3.99,Liter,Metro
```

Erste Zeile = Kopfzeile (wird übersprungen).
Trennzeichen: Komma oder Semikolon.
